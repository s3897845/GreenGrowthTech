GREEN GROWTH

Is a software and hardware solution consisting of a set of sensors combined into a single architecture, as well as a web application that displays yield data during the harvesting process. As a result, a client receives yield maps from each sq.m. of a field which helps to improve efficency of their fields and reduce fertilizing costs using the percise analytics provided.
